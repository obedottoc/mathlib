
# Scientific Calculator Project

## Installation & Usage

1. **Clone the repository**:
```bash
git clone <repository-url>
cd scientific-calculator
````

2. **Run the application**:

```bash
python App.py
```

3. **Use the CLI menu** to perform calculations and conversions.
